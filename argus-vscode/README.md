# Argus — VS Code extension

Умный роутер LLM с экономией. Вставляешь свой API-ключ (Anthropic / OpenAI / Gemini),
Argus сам подбирает **самую дешёвую подходящую модель** под задачу, а в статус-баре внизу
показывает счётчик `потрачено / бюджет` и сколько ты **сэкономил** относительно премиум-модели.

## Запуск (локально, F5)

1. Открой **именно эту папку** (`argus-vscode`) в VS Code: `File → Open Folder…`
2. Первый раз: установи зависимости — в терминале `npm install`
3. Нажми **F5** (или `Run → Start Debugging`). Откроется второе окно — *Extension Development Host*.
4. Внизу справа появится `⚡ Argus 0 / 10000`.

## Команды (`Ctrl+Shift+P`)

| Команда | Что делает |
|---|---|
| **Argus: Run Prompt (auto-pick model)** | спросит запрос → роутер выберет модель → обновит счётчик. Без ключа работает в режиме симуляции, чтобы сразу видеть, как считается экономия. |
| **Argus: Set API Key** | сохранить ключ провайдера (в безопасном хранилище VS Code SecretStorage). |
| **Argus: Clear API Key** | удалить ключ. |
| **Argus: Show Usage / Savings** | сводка + кнопки (клик по статус-бару). |
| **Argus: Reset Usage Counter** | обнулить счётчик. |

## Настройки

- `argus.budgetCents` — бюджет в центах USD (по умолчанию `10000` = $100). Показывается в статус-баре как `spent / budget`.

## Как считается экономия

1. `estimateTier()` грубо оценивает сложность запроса → `cheap | mid | premium`.
2. `pickModel()` среди подключённых провайдеров берёт **самую дешёвую** модель нужного тира.
3. Стоимость = токены × цена модели; экономия = (стоимость на премиум-модели) − (фактическая).

Цены в `src/router.ts` — приблизительные публичные, легко поправить под реальные.

## Структура

```
src/
  extension.ts   точка входа: статус-бар + команды
  providers.ts   вызовы Anthropic / OpenAI / Gemini (global fetch)
  router.ts      каталог моделей, выбор тира и самой дешёвой модели, расчёт стоимости
  usage.ts       счётчики расхода/экономии (globalState) + бюджет
  secrets.ts     хранение API-ключей в SecretStorage
```

> Это рабочий каркас v1. Реальные вызовы провайдеров включаются автоматически, как только добавлен ключ; иначе — симуляция.
